# CHG-20260517-010 添加人物角色名必填校验

## 背景

创建系列和系列详情/角色审核里都支持添加人物。角色名是后续剧本解析、角色资产生成、分镜人物引用的关键字段，不能允许空值进入提交流程。

## 改动

- 创建系列页动态角色卡的角色名称输入框增加 `required` 和必填提示标记。
- 创建系列提交前逐张角色卡校验角色名，空角色名会立即提示并聚焦到对应输入框。
- 系列详情/角色审核的添加角色弹窗中，角色名称输入框增加 `required` 和必填提示标记。
- 角色审核新增角色提交时，空角色名会使用统一的必填聚焦提示，不再只弹出普通提示后停留在原位置。
- 新增模板测试，约束两个入口都必须保留角色名必填校验。

## 影响范围

- 创建系列页 `series_init.html` 的添加角色卡片与提交校验。
- 系列详情/角色审核页 `series_review.html` 的添加新角色弹窗。
- 不改变角色生成接口、积分扣费、手动上传人物图和角色审核流程。

## 验证

- `python manage.py test apps.series.tests.RoleNameGuidanceTests`
- `python manage.py check`

## 重启

- 已重启前端 Django 服务，监听地址 `127.0.0.1:8000`。
- 已重启后端 Spring Boot 服务，监听地址 `127.0.0.1:8081`。
