# CHG-20260510-024 剧集详情角色资产增加修改入口

## 背景

剧集详情页可以查看角色资产，但没有明显入口返回角色审核页修改角色。用户需要从首页重新找到系列，操作路径偏长。

## 改动

1. 在剧集详情页“角色资产”标题右侧新增“修改角色”按钮。
2. 按钮跳转到当前系列的角色审核/修改页：`/<series_id>/review/`。
3. 每张角色卡片右上角新增悬浮编辑入口，跳转到 `/<series_id>/review/#role-<role_id>`，进入后自动滚动到对应角色。

## 影响范围

- `frontend/templates/episode/episode_detail.html`

## 验证

- `python3 manage.py check`
- 页面脚本语法检查通过。
- 前后端已重启。
