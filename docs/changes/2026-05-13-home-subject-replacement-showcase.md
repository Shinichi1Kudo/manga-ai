# CHG-20260513-011 首页主体替换营销展示区

## 背景

首页底部需要预留可扩展的营销功能展示位置，先用于展示“视频主体替换 - 精准定向修改”。后续会替换为用户提供的替换前视频、参考图、替换后视频和提示词。

## 改动

- 在首页“我的系列”下方新增功能展示区。
- 增加“视频主体替换 - 精准定向修改”的标题、说明和体验入口。
- 增加替换前视频、参考图、替换后视频三栏演示位。
- 预留“原视频中要替换的对象”和“替换成”两个提示词展示位。
- 当前素材使用默认 URL 占位，后续可直接替换为真实素材地址。

## 影响范围

- `frontend/templates/series/series_list.html`

## 验证

- `python3 manage.py check`
- `mvn -DskipTests compile`
- 重启前端 `8000` 与后端 `8081`
