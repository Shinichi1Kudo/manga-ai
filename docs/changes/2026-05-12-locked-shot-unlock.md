# CHG-20260512-009 已锁定分镜支持解锁回待审核

## 背景

已锁定分镜列表需要支持用户撤回锁定。解锁后分镜应回到待审核分镜列表底部，并恢复待审核分镜的编辑、生成、上传和删除等能力。

## 改动

- 后端新增分镜解锁接口：`POST /api/v1/shots/{shotId}/unlock/`。
- 解锁后后端将分镜状态从已锁定恢复为待审核。
- 如果剧集已经因为全部分镜锁定而标记完成，解锁任一分镜时同步恢复剧集为待审核。
- 已锁定分镜列表中新增“解锁分镜”按钮。
- 解锁成功后前端不刷新页面，直接把卡片移动到待审核分镜列表底部。
- 用户自定义分镜标题会保留；未自定义标题的分镜按回到待审核列表后的顺序显示默认 `分镜N`。

## 影响范围

- 后端：分镜服务、控制器与 Django 转发接口。
- 前端：剧集详情页已锁定分镜列表、待审核分镜列表迁移逻辑。
