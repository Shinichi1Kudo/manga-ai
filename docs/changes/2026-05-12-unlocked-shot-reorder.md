# CHG-20260512-010 解锁后待审核分镜拖拽排序修复

## 背景

已锁定分镜解锁后会回到待审核分镜列表，但拖拽调整顺序时后端仍按全剧集分镜数量校验。只要剧集里还存在其他已锁定分镜，前端提交的待审核分镜列表数量就会和全量分镜不一致，导致排序失败。

## 改动

- 后端排序接口改为只校验和更新待审核分镜集合。
- 已锁定分镜仍不参与排序，也不会被待审核列表拖拽影响。
- 前端排序保存失败时增加错误提示，避免用户只看到列表回弹。

## 影响范围

- 后端：分镜排序服务。
- 前端：剧集详情页待审核分镜拖拽排序失败提示。
