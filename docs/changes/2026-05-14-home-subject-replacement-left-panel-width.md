# CHG-20260514-004 首页主体替换展示区左侧说明栏收窄

## 背景

移除展示区底部标签后，上一版通过拉伸视频区域减少留白，但视觉上会让替换前视频、参考图和替换后视频显得被硬拉高。现在改为收窄左侧说明栏，释放右侧展示区宽度，同时保持媒体原始比例。

## 改动

- 左侧 `Precision Edit` 说明栏宽度从 `360px` 调整为 `280px`。
- 左侧说明栏内边距、标题字号和信息卡片间距同步收紧。
- 右侧展示区取消纵向拉伸布局。
- 替换前视频、参考图、替换后视频恢复固定比例展示：
  - 替换前视频：`aspect-video`
  - 参考图：移动端 `aspect-video`，桌面端 `4:5`
  - 替换后视频：`aspect-video`

## 影响范围

- 前端模板：`frontend/templates/series/series_list.html`
- 不涉及后端接口和数据库。

## 验证

- `python3 manage.py check` 通过。
- `mvn -DskipTests compile` 通过。
- `git diff --check` 通过。
- 模板关键字检查确认左侧说明栏为 `280px`，右侧展示区不再保留上一版拉伸类，媒体区域恢复固定比例。
- 前端 `8000`、后端 `8081` 已重启并确认监听。
