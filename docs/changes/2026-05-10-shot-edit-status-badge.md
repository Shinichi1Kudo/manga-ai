# 分镜列表编辑状态标记

## 编号

CHG-20260510-010

## 背景

分镜列表里之前没有明确展示哪些分镜被用户改过，用户只能点开内容自己判断，容易混淆系统自动解析内容和人工调整内容。

## 改动

- 在每个分镜卡片标题区新增编辑状态徽标：
  - `已编辑`：剧情或场景任意一个被用户主动编辑过。
  - `未编辑`：剧情和场景都没有用户编辑标记。
- 徽标放在生成状态标签旁边，便于扫列表时快速判断。
- 徽标带 hover 提示：
  - 已编辑：`场景或剧情已手动编辑`
  - 未编辑：`场景和剧情未手动编辑`
- 分镜资产轮询刷新时，会同步后端最新的 `descriptionEdited` / `sceneEdited` 状态并更新徽标。
- 用户在页面内修改剧情或场景并保存成功后，徽标会立刻切换为 `已编辑`。

## 判定规则

```text
descriptionEdited == true 或 sceneEdited == true => 已编辑
descriptionEdited == false 且 sceneEdited == false => 未编辑
```

## 验证

- `python3 manage.py check` 通过。
- 前端 `8000` 已重启。
- 后端 `8081` 监听正常。
- 接口抽样确认分镜可返回 `descriptionEdited` 和 `sceneEdited` 状态。

## 影响范围

- 剧集详情页分镜列表。
- 分镜剧情/场景内联编辑后的状态展示。
