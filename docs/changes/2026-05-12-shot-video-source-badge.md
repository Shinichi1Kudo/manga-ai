# CHG-20260512-005 待审核分镜视频来源标记

## 背景

待审核分镜支持手动上传视频后，用户无法在分镜列表里直接判断当前视频是手动上传的版本，还是系统生成的版本。

## 改动

- `ShotDetailVO` 增加 `videoSource` 字段，取值为 `manual` 或 `system`。
- 分镜列表接口批量读取当前激活视频资产及其元数据，识别 `manual-upload` 为手动上传，其余已有完成视频识别为系统生成。
- 待审核分镜卡片在已完成且有视频时显示来源标记按钮：
  - `手动上传`
  - `系统生成`
- 手动上传成功后，当前页面立即把分镜标记更新为 `手动上传`。
- 分镜进入生成中或移动到已锁定列表时移除待审核来源标记。

## 影响范围

- 后端：分镜详情 VO 与分镜列表/详情数据组装。
- 前端：剧集详情页待审核分镜列表展示。
