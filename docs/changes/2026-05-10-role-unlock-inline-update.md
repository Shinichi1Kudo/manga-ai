# CHG-20260510-025 角色解锁改为局部更新

## 背景

系列角色审核页点击“解锁”后，后端接口很快返回，但前端会延迟 1 秒并整页刷新。刷新会重新加载角色审核模板、资产提示词和用户信息，导致体感等待数秒。

## 改动

1. 解锁成功后不再执行 `location.reload()`。
2. 前端直接把当前角色卡片更新为“待审核”状态。
3. 立即恢复重新生成、设为默认、生成新服装、删除等按钮的可用状态。
4. 同步更新角色统计和“下一步”按钮状态。

## 影响范围

- `frontend/templates/series/series_review.html`

## 验证

- `python3 manage.py check`
- 页面脚本语法检查通过。
- 前后端已重启。
