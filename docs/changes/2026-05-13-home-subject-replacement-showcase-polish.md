# CHG-20260513-012 首页主体替换展示区视觉优化

## 背景

首页底部“视频主体替换 - 精准定向修改”展示区第一版信息完整，但视觉上偏普通卡片堆叠，不够像一个可对外展示的功能模块。

## 改动

- 将展示区改为更清晰的产品演示模块，顶部增加功能聚焦标题和独立入口。
- 优化 `BEFORE / REFERENCE / AFTER` 三段演示的排版、边框、状态标签和层级。
- 将提示词占位区调整为更明确的“原视频中要替换的对象 / 替换成”输入信息展示。
- 优化首页“主体替换”按钮里的 `NEW` 徽标，让它更轻、更精致，保留英文文案。
- 继续保留默认视频、参考图和 `xxx` 提示词占位，方便后续替换真实素材。

## 影响范围

- `frontend/templates/series/series_list.html`

## 验证

- `python3 manage.py check`
- `mvn -DskipTests compile`
- 重启前端 `8000` 与后端 `8081`
- 浏览器打开首页检查展示区渲染
