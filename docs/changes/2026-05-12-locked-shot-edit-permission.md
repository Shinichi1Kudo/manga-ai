# CHG-20260512-008 已锁定分镜编辑权限收紧

## 背景

已锁定分镜代表已经审核通过并进入锁定列表。锁定后只应该允许用户调整分镜标题，剧情、场景、视频参数、视频版本和生成状态都不能再被修改。

## 改动

- 前端已锁定分镜卡片只保留分镜标题输入框可编辑。
- 已锁定分镜隐藏或禁用以下入口：
  - 拖拽排序
  - 剧情与场景编辑
  - 时长、分辨率、模型、比例下拉框
  - 生成、重新生成、上传视频、锁定、删除按钮
  - 视频历史里的回滚按钮
- 后端增加锁定态校验，锁定分镜只允许纯标题更新。
- 后端拦截锁定分镜的重新生成、手动上传视频、参考图更新、视频回滚、删除、重复审核和排序修改。
- 批量生成视频时跳过已锁定分镜，避免被后台任务重新进入生成流程。

## 影响范围

- 前端：剧集详情页待审核分镜列表与已锁定分镜列表。
- 后端：分镜更新、审核、生成、上传、参考图、视频版本回滚、删除、排序与批量生成逻辑。

## 验证

- 已通过：`node --check` 剧集详情页脚本抽取结果
- 已通过：`python3 manage.py check`
- 已通过：`mvn -DskipTests compile`
- 已完成：前后端重启
