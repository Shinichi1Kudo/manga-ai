# CHG-20260514-005 首页主体替换说明文案下移

## 背景

首页“视频主体替换-精准定向修改”标题下方的说明文案让标题区显得偏挤，而视频展示区下方在移除功能标签后仍有一定空白。

## 改动

- 移除标题 `视频主体替换 - 精准定向修改` 下方的说明文案。
- 将说明文案移动到右侧视频展示区下方。
- 新说明区域使用轻量信息条样式，填补视频下方空白，同时不拉伸视频比例。

## 影响范围

- 前端模板：`frontend/templates/series/series_list.html`
- 不涉及后端接口和数据库。

## 验证

- `python3 manage.py check` 通过。
- `mvn -DskipTests compile` 通过。
- `git diff --check` 通过。
- 模板关键字检查确认标题仍保留，说明文案只在视频展示区下方出现一次。
- 前端 `8000`、后端 `8081` 已重启并确认监听。
