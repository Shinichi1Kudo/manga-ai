# CHG-20260512-006 待审核分镜生成耗时文案增强

## 背景

待审核分镜列表里，系统生成视频完成后原本只显示一个耗时数字，例如 `12秒`，用户不容易理解这是本次系统生成视频的耗时。

## 改动

- 将系统生成视频的耗时展示改为明确文案：`本次视频生成耗时`。
- 耗时数字单独高亮显示，避免和其他时间信息混淆。
- 手动上传视频不展示系统生成耗时，避免把上传版本误认为模型生成结果。

## 影响范围

- 前端：剧集详情页待审核分镜列表左侧操作区。

## 验证

- 已通过：`python3 manage.py check`
- 已通过：`node --check` 剧集详情页脚本抽取结果
- 已完成：前后端重启
