# CHG-20260514-002 首页主体替换演示视频控制增强

## 背景

首页“视频主体替换-精准定向修改”展示区的替换前、替换后视频仍有暗色视觉蒙层，影响观看素材细节。同时视频缺少直接的声音开关和放大查看入口。

## 改动

- 移除替换前视频和替换后视频上的暗色/渐变蒙层。
- 去掉视频元素本身的半透明样式，画面按原始亮度展示。
- 两个演示视频默认保持静音自动播放。
- 为两个演示视频增加声音切换按钮：
  - 默认显示静音图标，提示“开启声音”。
  - 开启声音后切换为音量图标，提示“静音”。
- 为两个演示视频增加放大按钮。
- 新增放大视频弹窗，使用原视频地址播放，并保留浏览器原生播放控制。

## 影响范围

- 前端模板：`frontend/templates/series/series_list.html`
- 不涉及后端接口和数据库。

## 验证

- `python3 manage.py check` 通过。
- `mvn -DskipTests compile` 通过。
- `git diff --check` 通过。
- 模板关键字检查确认替换前/替换后视频已挂载 `data-showcase-video`，并存在声音切换、放大按钮和放大视频弹窗。
- 模板关键字检查确认替换前/替换后视频不再使用 `opacity-65`、`opacity-70` 和原替换后径向暗色蒙层。
- 前端 `8000`、后端 `8081` 已重启并确认监听。
